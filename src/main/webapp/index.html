What is this site about?



If you were looking for site filled with different programs for both front- and back-end development on Windows - you are in the right place. We have different setups from different users on different Windows versions starting from XP. You can see how easy it is to find an IDE or any other tool exactly for your needs. You can also create an account and post your setup.


Communication

Also check our forum, where you can find info on how to install or edit any program used in any setup. If you haven't found what you need to or wanted program's topic doesn't exist - feel free to ask a question.


All for one and one for all

You can use your account here to log into your forum account even though you haven't created one! It's much more comfortable and simple to remember one password for two services, isn't it?
